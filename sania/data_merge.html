Data

Load raw data

We use two datasets:

  • V-Dem (Varieties of Democracy): country-year measures of democracy, governance, and civil liberties
  • OECD DAC2A: total ODA (Official Development Assistance) disbursements by recipient country and year, 1960–2024
vdem <- read_csv(here("VDEM.csv"))
oecd_raw <- read_csv(here("sania/full_oecd_D.csv"))

Clean and aggregate OECD data

Country names differ between the two datasets, so we standardize OECD recipient names to match V-Dem before merging. We then aggregate from donor-level rows to total ODA received per recipient country per year.

oecd_clean <- oecd_raw %>%

Merge V-Dem and OECD

We merge on country_name and year, keeping all V-Dem observations (left join). We also compute delta_polyarchy — the year-over-year change in the electoral democracy index — before subsetting years, so that the first observation in the analysis window can draw its lag from the prior year.

  • Negative delta_polyarchy = democratic backsliding
  • Positive delta_polyarchy = democratization
merged <- vdem %>%
  left_join(oecd, by = c("country_name", "year")) %>%
  arrange(country_name, year) %>%
  group_by(country_name) %>%
  mutate(delta_polyarchy = v2x_polyarchy - lag(v2x_polyarchy)) %>%
  ungroup()

Subset to analysis variables (1960–2024)

We keep only the variables needed for analysis and filter to the period covered by the OECD data.

Add GDP per capita and population

GDP per capita and population are not in our V-Dem extract, so we pull them from the World Bank via the WDI package.

wdi <- WDI(
  indicator = c(gdp_per_capita = "NY.GDP.PCAP.KD",
                population     = "SP.POP.TOTL"),
  start = 1960, end = 2024,
  extra = FALSE
) %>%
  mutate(country_name = countrycode(iso2c, origin = "iso2c",
                                    destination = "country.name")) %>%
  filter(!is.na(country_name)) %>%
  dplyr::select(country_name, year, gdp_per_capita, population)

analysis <- analysis %>%
  left_join(wdi, by = c("country_name", "year"))

Prepare outcome variable

ODA is reported in USD millions (continuous). We convert it to an integer count by rounding to the nearest million and flooring any negative values (loan repayments) to zero, as required by count models.

analysis <- analysis %>%
  mutate(oda_count = pmax(0, round(total_oda_usd_millions))) %>%
  dplyr::select(-total_oda_usd_millions) %>%
  filter(!is.na(oda_count), !is.na(v2x_polyarchy))


write_csv(analysis, here("sania/analysis_merged.csv"))

Models

Model strategy

We use negative binomial regression to test whether democratic backsliding predicts foreign aid received. ODA is heavily overdispersed — its variance far exceeds its mean — which rules out Poisson. Population is included as a control (log(population)).

library(MASS)

Overdispersion check

Under Poisson, the variance should equal the mean. If the variance-to-mean ratio is much greater than 1, the data is overdispersed and negative binomial is more appropriate.

tibble(
  mean     = mean(analysis$oda_count, na.rm = TRUE),
  variance = var(analysis$oda_count,  na.rm = TRUE),
  ratio    = var(analysis$oda_count,  na.rm = TRUE) / mean(analysis$oda_count, na.rm = TRUE)
)

The variance-to-mean ratio is far greater than 1, confirming severe overdispersion. We use negative binomial regression, which adds a dispersion parameter to account for this.

Backsliding model: Democratic backsliding and aid (Negative Binomial)

The key predictor is delta_polyarchy (year-over-year change in v2x_polyarchy). We keep v2x_polyarchy as a level control so that delta_polyarchy captures the effect of change net of overall democratic standing. The expected sign on delta_polyarchy is positive — backsliding (negative delta) should predict less ODA.

nb_backsliding <- glm.nb(
  oda_count ~ delta_polyarchy + v2x_corr + v2x_rule + v2xcl_rol +
              log(gdp_per_capita) + log(population),
  data = analysis
)

Robustness Checks

Check 1: Alternative democracy measures

v2x_polyarchy measures electoral democracy specifically. If the null result is real, it should hold across different conceptualizations of democracy. We test two alternatives:

  • v2x_libdem — liberal democracy (adds judicial constraints and civil liberties)
  • v2x_partipdem — participatory democracy (emphasizes civil society and direct participation)

Check 2: Extended time window

The analysis now uses the full OECD time series (1960–2024), providing a much longer panel with more backsliding episodes and greater statistical power compared to the original 2010–2020 window. Year coverage in practice is limited by the availability of V-Dem scores and World Bank GDP/population data, but the full window is retained and listwise deletion handles missing observations.

Replication: Commitment Data

ODA commitments capture when donors pledge aid, while disbursements capture when it is actually transferred. Commitments may respond more quickly to political changes since they are forward-looking decisions, whereas disbursements reflect prior commitments and are stickier. We replicate the main analysis using OECD DAC3A commitment data (1966–2024) to test whether the null result on backsliding holds.
